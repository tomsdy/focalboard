Focalboard: Project Tracking for Software Development Teams

Focalboard helps your software development team stay aligned to complete tasks, reach milestones, and achieve goals. Plan every project and hit every milestone with this open source alternative to tools like Asana, Trello, and Notion.

For Teams: Check out Mattermost Boards with integrated collaboration, for cloud or self-hosted.

For Individuals: Get Personal Desktop, a stand-alone single-user app for Mac, Windows, and Linux.

For Developers: Fork the source code on GitHub and install personal server for development.

Focalboard is open source! Check out the source code on GitHub, and contribute to the future of this project. Join the discussion on the Mattermost community to interact with other Mattermost and Focalboard contributors and users.

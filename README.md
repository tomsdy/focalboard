# mattermost-octo-tasks

## Build instructions

```
npm i
make
```

Currently tested with:
* Go 1.15.2
* MacOS Cataline (10.15.6)

## Running and testing

To start the server:
```
./bin/octoserver
```

Server settings are in config.json.

Open a browser to [http://localhost:8000](http://localhost:8000) to start.
